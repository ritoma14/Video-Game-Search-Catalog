# Video Game Search Catalog


## Requirements
- Javascript
- Node.js
- Express.js 
- PostgreSQL
- Socket.io
- Python3



## Recommended Tools

- [VSCode Editor](https://code.visualstudio.com/download)
 OR
- [Any Text Editor]



## Installing Node.js
Terminal command line:
```
$ sudo apt install nodejs
```

## Install npm

```
$ sudo apt install npm
```


## Install PostgreSQL interface for Node.js

```
$ npm i pg-promise
```

## Install Nodejs Express

```
$ npm i express
```

## Install Socket.io

```
$ npm i socket.io
```

## Install python 3
Need python3 in order to install psycopg, a popular PostgreSQL database adapter for python
```
$ sudo apt install python3-psycopg2
```
## Install PostgreSQL
Install PostgreSQL in current directory
```
$ sudo apt install postgresql
```
